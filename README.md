
# Rational Vacuum Cleaner Agent with PyQt5

![Vacuum Agent](cleaner.jpg)

**تطبيق رسومي تفاعلي لوكيل مكنسة كهربائية عقلاني (Rational Agent) في بيئة متاهة مع عوائق وتسخ ديناميكي**

مشروع تعليمي ممتاز لفهم مفاهيم الوكلاء العقلانيين في الذكاء الاصطناعي، مبني بواجهة رسومية جميلة باستخدام **PyQt5**.

### المميزات الرئيسية
- خريطة 10×10 مع عوائق (جدران زرقاء)
- خلايا تتسخ تدريجيًا باحتمالية محددة (Dynamic Dirt)
- وكيل ذكي يتحرك وينظف مع **ذاكرة بسيطة** (يتجنب الجدران التي اصطدم بها سابقًا)
- شكل الوكيل مثلث يتغير حسب الاتجاه (أعلى/أسفل/يمين/يسار)
- لون أحمر عند الاصطدام (Bump)
- تدرج لوني للخلايا حسب درجة التسخ
- دعم تحميل خريطة من ملف خارجي (.map)
- تقييم الأداء:
  - درجة التسخ المتبقية (Dirty Degree)
  - الطاقة المستهلكة (Consumed Energy)
  - متوسط الأداء عبر عدة جولات
- أزرار تحكم كاملة:
  - خطوة واحدة، جولة واحدة، كل الجولات
  - تغيير عدد الجولات ومدة الحياة (Life Time)
  - إعادة التشغيل

### طريقة التشغيل
1. تأكد من تثبيت PyQt5:
   ```bash
   pip install PyQt5
   ```
2. ضع الملفات التالية في نفس المجلد:
   - `vacuum_agent.py` (ملف الكود الرئيسي)
   - `cleaner.jpg` (أيقونة الوكيل - اختياري)
   - ملف خريطة مثل `agent.map`
3. شغّل البرنامج:
   ```bash
   python vacuum_agent.py
   ```
